# Ulti-Stats-Analysis
Power BI Dashboard providing performance analysis
Ulti Stats Analysis 📊🥏

A data analytics and visualization project focused on evaluating ultimate frisbee player and team performance through interactive dashboards and advanced statistical tracking.

**Overview**

Ulti Stats Analysis is a performance analytics dashboard designed to transform raw ultimate frisbee data into actionable insights. The project tracks both traditional and advanced metrics to help evaluate player efficiency, team effectiveness, and in-game performance trends.

Built with a focus on clean visualizations and meaningful analytics, this dashboard allows users to analyze offensive and defensive impact, playing time distribution, point efficiency, and overall team performance.

Inspired by modern sports analytics platforms and ultimate frisbee stat tracking tools such as UltiStats and community-driven ultimate analytics projects.

**Features**

**Player Performance Analytics**

-Goals, assists, and defensive blocks tracking
-Goals per point
-Assists per point
-Ds (blocks) per point
-Offensive efficiency metrics
-Defensive efficiency metrics
-Playing time analysis

**Team Analytics**
-Offensive vs defensive production
-Point differential trends
-Line efficiency tracking
-Possession-based insights
-Team scoring splits

**Interactive Dashboarding**
-Dynamic Power BI visualizations
-CSV-based data importing
-Filtering by player, game, or matchup
-Trend analysis and comparative statistics

**Tech Stack**
-Power BI — Dashboard creation & visualization
-CSV Data Sources — Data ingestion and storage
-Data Modeling — Relationships, calculated metrics, and performance KPIs
-Sports Analytics Concepts — Efficiency metrics, point-based normalization, and lineup analysis

*Metrics Tracked*
Category	                          Metrics

Defensive Stats	                   Ds (Blocks), Defensive Efficiency
